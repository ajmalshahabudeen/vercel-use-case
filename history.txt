- Added [run-docker.bat](apps/vercel-env-updater/run-docker.bat) in the app directory. This batch script:
  - Verifies if Docker is running.
  - Checks if the Postgres database container (`vercel-db`) is running/stopped/non-existent.

  - Rebuilds and launches the `vercel-env-updater` container on port 3001.
